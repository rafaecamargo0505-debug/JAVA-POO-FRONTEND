PDV Posto de Combustível — Frontend (Java Swing)

Aplicação desktop desenvolvida em Java Swing para interação com o sistema de PDV de um posto de combustível.
Esse frontend consome a API do backend e oferece uma interface simples e funcional para operadores, frentistas e gerentes.

✅ Visão Geral

A aplicação permite:

-Login com controle de acesso
-Registro de abastecimentos
-Abertura e fechamento de caixa
-Cadastro de clientes
-Vendas de produtos
-Consulta de saldos e contas de crédito
-Relatórios básicos
-Interface simples, direta e funcional

✅ Tecnologias Utilizadas

-Java 17
-Java Swing
-Maven
-HTTPClient para requisições
-ORMs e lógica estrutural em classes Java

✅ Arquitetura

O projeto segue uma organização intuitiva:

ui/        -> Telas (Swing)
service/   -> Comunicação com o backend
model/     -> Classes e entidades do sistema
util/      -> Auxiliares


✅ Pré-requisitos

Instale:

-JDK 17+
-Maven
-IDE Java (IntelliJ/Eclipse)

E claro:
✅ O backend precisa estar rodando antes do frontend.


✅ Como Executar o Frontend:

1-Abra o projeto na IDE

2-Aguarde baixar dependências

3-Execute a classe:
MainApp.java

A tela de login será exibida.
